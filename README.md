# بنيان - صفحة الهبوط

صفحة هبوط ثابتة لنظام إدارة المقررات الدراسية.

## النشر على GitHub Pages

1. أنشئ مستودع جديد على GitHub
2. ارفع ملفات هذا المجلد (`index.html`, `styles.css`, `README.md`)
3. اذهب إلى **Settings** > **Pages**
4. اختر **Source**: `Deploy from a branch`
5. اختر **Branch**: `main` والمجلد `/ (root)`
6. اضغط **Save**

سيتم نشر الموقع على الرابط:
```
https://[username].github.io/[repository-name]/
```

## تخصيص الصفحة

- لتغيير رقم الواتساب: عدّل الرابط في قسم "تواصل عبر واتساب"
- لتغيير البريد الإلكتروني: عدّل الرابط في قسم "راسلنا بالبريد"
- لتغيير الألوان: عدّل المتغيرات في `:root` في ملف `styles.css`
